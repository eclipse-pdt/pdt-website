PHP Perspective

The PHP Perspective is PDT 's default perspective. It incorporates all PDT 's PHP project/file creation, inspection and editing functionality.

The PHP Perspective contains the following views:

  • PHP Explorer View

  • Outline View

  • Type Hierarchy View

  • Problems View  

  • Tasks View

  • Console View
