Remote System Explorer

 

RSE (Remote System Explorer) provides transparent access to remote resources, including the upload and download of files. RSE helps you to create, view and manage your connections to remote systems such as FTP and SFTP. This will allow you to work on projects with local files remotely enabling you to create, view and manage your connections to remote systems.

RSE enables you to perform the following tasks:

 

• Create a connection via the Server’s view.

• Create a new remote project.

• Import a project from an existing connection.

• Share a local project to a connection.

• Synchronize a local project to a connection

• Export Actions

• Import Actions

 

RSE is an upgraded version of the Remote Systems View that reduces the amount of communication between networks, by using less connection nodes. RSE also communicates less data, by directly transferring information rather than copying all of the files.
