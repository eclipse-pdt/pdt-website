PHP Java Bridge Properties

The PHP Java Bridge Properties page allows you to set the Java Runtime Environment for the project.

 

To access the PHP Java Bridge Properties page, right-click a PHP project in PHP Explorer view and select Properties | Resource -or- select the project and from the menu bar go to Project | Properties | PHP Java Bridge Properties.

 

See Changing Projects' JREs for more information.
