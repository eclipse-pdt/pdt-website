Troubleshooting Zend Platform Integration

 

Your Zend Platform Integration might not function correctly for one of the following reasons:

  • Your Zend Platform server is not correctly configured in PDT.
    See Defining a Zend Platform Server in Studio for more information.

  • Zend Platform is situated behind a firewall or other security device.
    See Setting Up Tunneling for more information.

  • Your PDT Communication is not correctly configured in Zend Platform.
    See Configuring Studio Communication Settings in Platform for more information.

  • Zend Platform is situated on an HTTPS Server. In this case, when trying to connect to your Platform server from the Platform Events view, you will receive the following message:
    "Unable to verify the identity of 'https://<MyServer>/ZendPlatform' as a trusted site".
    See Troubleshooting Platform Integration on an HTTPS Server for more information.
