What's New in PDT 2.x

The following new features are available in PDT 2.2:

  • Templated Content Assist

  • Content Assist in PHPDoc Block

  • Syntax Coloring Scheme Editor - The Syntax Coloring Scheme editor is now conformed to Eclipse standards.

  • Advanced Syntax Coloring Rules

  • Show Inherited Members in the Quick Outline view - The Quick Outline view includes a "show inherited members" mode which lists all methods inherited by super types. For more information see Using Quick views in the Java Development User Guide.

  • More decorative Hover Information

  • Inspect Action for Debugging

  • Drag and Drop in PHP/JavaScript Editors

  • Powerful PHP Search dialog

  • Eclipse Marketplace

 

The following new features are available in PDT 2.1:

  • Phar Integration

  • Mylyn Integration

  • PHP 5.3 Support:

    • Ability to create a PHP project based on PHP 5.3.

    • Syntax Coloring and Validation for PHP 5.3 in the editor

    • Content Assist for Namespaces

    • Content Assist for packages

    • Namespace grouping in PHP Explorer

The following new features are available in PDT 2.0:

  • PHP Build Path Configuration

  • Override Indicators

  • Type Hierarchy

  • Mark Occurrences

  • Open Type/Method

  • Content Assist Enhancements - Camel Case matches, Magic Members support
